## Why this ADR is intentionally empty

This decision — the protocol for consolidating memory artefacts from a consultant's laptop to a company server under GDPR and EU AI Act constraints — will be drafted **jointly with an external governance reviewer** (Otopoetic, agreed 2026-04-18). The consent ledger and jurisdiction-routing semantics carry obligations that sit above the architectural layer; drafting them unilaterally would defeat the purpose of the collaboration.

The number is reserved here so references from other documents (`docs/roadmap.md` Phase 2.1, `docs/pitch/collaboration-roadmap-otopoetic.md`, `docs/architecture/product-and-dependencies.md`) resolve. The formal sections — Context / Decision / Consequences / Cross-references — will be populated through the co-authoring process, not filled in now.

Not filling this in now is itself a decision, and the right one.
